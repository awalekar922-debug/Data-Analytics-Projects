# Data-Analytics-Projects
## 1. Comparative Financial Analysis — India's Top 5 IT Firms
**Tools:** NeonDB | SQL | Excel | Pivot Tables  
**Data Source:** Screener.in

### Study Statement
Comparative financial analysis of India's Top 5 IT companies 
(TCS, Infosys, Wipro, HCL Tech & Tech Mahindra) over FY2019–2024, 
examining revenue & profit growth trends, financial health indicators 
and competitive positioning.

### Key Findings
- TCS is the consistent sector leader across all metrics
- HCL Tech is the strongest growth challenger in FY2024
- Tech Mahindra showed 51% profit decline in FY2024
- Entire sector maintains near-zero debt levels
- FY2022 marked a universal post-COVID earnings peak
